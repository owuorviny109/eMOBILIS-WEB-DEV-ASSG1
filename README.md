# Bonkers Cat Food Website

A modern, responsive multi-page website for Bonkers Cat Food, a premium cat food brand based in Nairobi, Kenya. Features a clean design with custom CSS styling and comprehensive product information.

**Live Site:** https://owuorviny109.github.io/eMOBILIS-WEB-DEV-ASSG1/

## Project Structure

```
ESSENTIALS/
├── index.html              # Homepage with hero section and testimonials
├── our-food.html           # Product showcase with all available flavors
├── ingredients.html        # Detailed ingredient breakdown and benefits
├── stories.html            # Customer testimonials and success stories
├── help.html               # Quick help and FAQ preview
├── get-bonkers.html        # Product ordering page with pricing
├── about.html              # Company story and mission
├── contact.html            # Contact information and location
├── faq.html                # Comprehensive frequently asked questions
├── feeding-guide.html      # Portion sizes and feeding recommendations
├── nutritional-info.html   # Detailed nutritional facts and analysis
├── main.css                # Custom CSS styling for all pages
└── README.md               # Project documentation
```

## Features

- Multi-Page Architecture: Eleven interconnected pages with consistent navigation
- Responsive Design: Mobile-first approach with breakpoints for tablets and desktop
- Modern Layout: Clean grid-based layout with semantic HTML structure
- Custom CSS: No framework dependencies - pure CSS implementation
- Consistent Branding: Unified header, footer, and styling across all pages
- Accessibility: Proper alt texts and semantic markup
- Local Context: Kenyan locations, pricing in KSh, and local delivery information

## Page Descriptions

### index.html
Homepage featuring a hero section with the main value proposition, veterinarian testimonials with paw print decorations, and customer review cards. Sets the tone for the brand with warm, inviting design.

### our-food.html
Product showcase page displaying all four available flavors: Wild Salmon, Free-Range Chicken, Turkey & Cranberry, and Grass-Fed Beef. Highlights what makes each formula unique and the brand's differentiators.

### ingredients.html
Comprehensive breakdown of ingredients used in Bonkers formulas. Features two sections: what goes into every bowl (beneficial ingredients) and what you'll never find (excluded ingredients). Educational and transparent.

### stories.html
Customer testimonials page featuring real stories from cat owners across Nairobi. Includes reviewer names, locations, and profile images to add authenticity and social proof.

### help.html
Quick help page with frequently asked questions preview and links to more detailed resources. Serves as a gateway to support information with clear call-to-action buttons.

### get-bonkers.html
Product ordering page with pricing in Kenyan Shillings, product descriptions, and add-to-cart functionality. Includes delivery information and subscription options with savings.

### about.html
Company story page explaining the brand's mission, values, and commitment to quality. Features the Nairobi origin story and four core pillars: local sourcing, science-backed formulas, sustainability, and cat-first approach.

### contact.html
Contact information page with email, phone number, physical address in Westlands, Nairobi, and social media handles. Includes business hours in East Africa Time.

### faq.html
Comprehensive FAQ page covering twelve common questions about transitioning cats to Bonkers, product suitability, manufacturing location, guarantees, ingredients, allergies, delivery, and subscriptions.

### feeding-guide.html
Detailed feeding recommendations organized by cat age groups: kittens, young cats, adults, and seniors. Includes daily portion sizes, meal frequency, and helpful feeding tips.

### nutritional-info.html
Complete nutritional analysis page featuring guaranteed analysis percentages for crude protein, fat, fiber, and moisture. Lists key nutrients including taurine, omega fatty acids, and calcium with their benefits.

### main.css
Custom CSS file providing consistent styling across all pages with:
- Responsive grid layouts for different screen sizes
- Typography hierarchy using Poppins font family
- Component styling for cards, buttons, and navigation
- Hover effects and smooth transitions
- Warm color palette with cream backgrounds and amber accents
- Dark mode support for all components

## Navigation Structure

All pages share a consistent header with navigation links to:
- Our Food
- Ingredients
- Stories
- Help

And a comprehensive footer with three columns:
- Product Details: Ingredients, Nutritional Info, Feeding Guide
- Quick Links: About Us, Our Mission, FAQ
- Contact Us: Email link and contact information

## Local Context

The website is tailored for the Kenyan market with:
- Pricing in Kenyan Shillings (KSh)
- Nairobi-based locations and neighborhoods
- Local phone number format (+254)
- East Africa Time (EAT) business hours
- Same-day delivery within Nairobi
- References to Kenyan farms and local sourcing
